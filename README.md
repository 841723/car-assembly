# Car Assembly

```prac5.s``` has all source code

## CONTROLS

Write in UART #2
- Player 1:
    - UP: I
    - DOWN: K
    - LEFT: J
    - RIGHT: L

- Player 2:
    - UP: W
    - DOWN: S
    - LEFT: A
    - RIGHT: D

#### SCREEN

Place memory on 0x40007E00, ASCII. Like this:

```
0x40007E00

0x40007E20

0x40007E40

0x40007E60

0x40007E80

0x40007EA0

0x40007EC0

0x40007EE0

0x40007F00

0x40007F20

0x40007F40

0x40007F60

0x40007F80

0x40007FA0

0x40007FC0

0x40007FE0
```
